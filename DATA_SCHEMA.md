# Схема данных favorites.json

## Общая информация

Файл `favorites.json` предназначен для хранения избранных элементов пользователей бота. 
В данной реализации бота для управления задачами, этот файл может использоваться для хранения:
- Избранных задач
- Шаблонов часто используемых задач
- Приоритетных/важных задач

## Структура данных

### Корневая структура

```json
{
  "user_123456": {
    "favorites": [...],
    "templates": [...],
    "last_updated": "2024-01-01T12:00:00.000Z"
  },
  "user_789012": {
    "favorites": [...],
    "templates": [...],
    "last_updated": "2024-01-01T13:30:00.000Z"
  }
}
```
```
Поля верхнего уровня
Поле            | Тип 	 |Обязательное    |Описание
____________________________________________________________________________________
user_id	        |string  |Да	          | ID пользователя ВКонтакте (ключ объекта)
____________________________________________________________________________________
favorites       |array	 |Да	          | Массив избранных задач пользователя
____________________________________________________________________________________
templates       |array	 |Нет             |Массив шаблонов задач
____________________________________________________________________________________________
last_updated	|string	 |Да	          |Дата и время последнего обновления (ISO 8601)
```
