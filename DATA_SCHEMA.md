# Схема данных favorites.json

## Общая информация

Файл `favorites.json` предназначен для хранения избранных пользователей, которых добавили пользователи бота. 
Файл создается автоматически в корневой директории проекта при первом добавлении в избранное.

## Структура данных

### Корневая структура

```
{
  "12345678": [  // ID пользователя бота ВКонтакте (ключ)
    {
      // избранный пользователь 1
    },
    {
      // избранный пользователь 2
    }
  ],
  "87654321": [  // Другой пользователь бота
    {
      // его избранные
    }
  ]
}
```
## Поля избранного пользователя
```
Поле            |Тип              |Обязательное |Описание                               |Пример
______________________________________________________________________________________________________________________
vk_user_id	|number           |Да	        |ID пользователя ВКонтакте        |123456789
______________________________________________________________________________________________________________________
first_name	|string           |Да	        |Имя                              |"Анна"
______________________________________________________________________________________________________________________
last_name	  |string           |Да	        |Фамилия	                        |"Смирнова"
______________________________________________________________________________________________________________________
profile_url	|string           |Да	        |Ссылка на профиль	              |"https://vk.com/id123456789"
______________________________________________________________________________________________________________________
added_at  	|string           |Да	        |Дата и время добавления (ISO 8601)	|"2024-01-15T14:30:00.123456"
______________________________________________________________________________________________________________________
city        |string или null  |Нет        |Город                              |"Москва"
______________________________________________________________________________________________________________________
age	        |number или null  |Нет        |Возраст                            |28
______________________________________________________________________________________________________________________
photos      |array            |Да         |Массив фотографий (макс. 3)        |[...]
______________________________________________________________________________________________________________________
```
## Поля фотографии 
```
Поле  	|Тип	  |Обязательное	|Описание	            |Пример
____________________________________________________________________________________________
id	    |number	|Да	        |ID фотографии	        |456789123
____________________________________________________________________________________________
owner_id|number	|Да	        |ID владельца	          |987654321
____________________________________________________________________________________________
likes	  |number	|Да	        |Количество лайков    	|45
____________________________________________________________________________________________
url    	|string	|Да	        |Прямая ссылка на фото	|"https://sun9-78.userapi.com/..."
____________________________________________________________________________________________
```

## Примечания по валидации
1. **Уникальность:** В рамках одного пользователя бота не может быть двух записей с одинаковым vk_user_id
2. **Фотографии:** Сохраняются только топ-3 фотографии по количеству лайков
3. **Даты:** Все даты хранятся в формате ISO 8601 для удобства сортировки
4. **Ссылки:** URL профиля формируется как https://vk.com/id{vk_user_id}
