# Дипломный проект по курсу «Python-разработчик с нуля»
## VKinder
### Цель проекта
***Цель проекта — разработать программу-бота для взаимодействия с API социальной сети ВКонтакте. Бот будет предлагать различные варианты людей для знакомств в социальной сети ВКонтакте в виде диалога с пользователем, сохраняя данные об избранных пользователях в файлах на диске.***

### Вам предстоит:

- разработать программу-бота на Python,
- спроектировать и реализовать систему хранения данных с использованием файлов (в формате JSON),
- настроить взаимодействие бота с ВКонтакте,
- написать документацию по использованию программы.
### В результате выполнения этого задания вы:

- получите практический опыт создания чат-бота,
- закрепите навыки работы с GitHub и программирования на языке Python,
- разработаете с нуля полноценный программный продукт, который можно будет добавить в портфолио Python-разработчика.

## Получение ключей API
### Токен сообщества
1. Зайдите в управление сообществом
2. Раздел "Работа с API"
3. Создайте ключ с правами: messages, photos, groups
4. Скопируйте ключ

### Сервисный ключ
1. Перейдите на https://vk.com/dev
2. Создайте приложение
3. В настройках приложения найдите "Сервисный ключ доступа"
4. Скопируйте ключ

## Решение проблем
### Бот не отвечает
1. Проверьте правильность токенов в .env
2. Убедитесь, что сообщество имеет права на отправку сообщений
3. Проверьте, запущен ли скрипт

### Не находятся анкеты
1. Проверьте сервисный ключ
2. Убедитесь, что в вашем профиле заполнены возраст и город
3. Возможно, в вашем городе мало пользователей

### Ошибки с фотографиями
1. Проверьте права доступа к фотографиям
2. Убедитесь, что у пользователя есть фотографии в профиле
